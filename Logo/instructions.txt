Main Elements
1. Typography

イニシャル

YT

文字は

Gothic
Vampire
Sharp
Serif

を融合したオリジナルデザイン。

既存フォントは使用しない。

SVG Pathで作成する。

2. Hexagon

背景は正六角形。

条件

線のみ
二重線
均等な余白
完全な左右対称
3. Music

ロゴ中央を

五線譜

が横断する。

含める要素

Staff
Treble Clef
Musical Notes

装飾は控えめ。

4. Vampire

吸血鬼要素は

コウモリ

のみ。

条件

上部中央に小型コウモリ
Y,Tの上部に翼を融合
可愛いよりスタイリッシュ

牙や血は使用しない。

エージェントへの追加指示
AI画像をトレースしないこと。 本プロジェクトは既存画像を参考イメージとしつつ、ゼロからオリジナルのベクターアートとして設計する。
SVGはソースコードとして保守可能であること。 id、コメント、グループ構造を整理し、GitHubで差分レビューしやすい記述にする。
ロゴはアイコン（32px程度）からA4印刷まで視認性を保つこと。 細線や装飾は縮小時の見え方を考慮して設計する。
コミット単位で制作すること。 例: feat: create hexagon base → feat: add typography → feat: add music layer → feat: add pcb decorations → refactor: simplify paths。
ライセンス面に配慮すること。 使用する図形・パス・フォント・シンボルはすべてオリジナルまたはライセンス上問題のないものとし、第三者の知的財産を侵害しないこと。
