## overview
制作チーム：プログラマー5名、プランナー2名、イラストレーター3名、3Dモデラ―1名
担当箇所：インゲームのゲームシステム全般、プログラマーのマネジメント

バンタンで開催された2日間のゲームジャムイベントで制作された「落ち物系」がテーマのゲームです。
降り注ぐアイテムや邪魔者を取得したり捨てたりしながら長く生き残るアクションゲームです。
# 説明
11人という大規模なチームで作成したため、素材の受け渡しやタスクの割り振りなどに苦労しました。
最終的にはイラストレーターと密に連携し、UIやステージ配置を早々に完成させた事で、実装に間に合わせる事ができました。
# アピールポイント
多彩な敵の動きと安定した物理挙動による、落ち物パズルならぬ落ち物バトル感が人気になりました。
