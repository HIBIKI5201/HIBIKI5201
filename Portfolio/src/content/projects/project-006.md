## overview
制作チーム：プログラマー2名、プランナー1名
担当箇所： インゲーム全て

KONAMI主催のIndie Games Contest 学生選手権 2024に応募した、2Dアクションゲームです。
剣術と射撃の二つのモードを切り替えながら、押し寄せる妖怪達を倒すアクションゲームです。
# 説明
純粋な2Dアクションバトルを現時点の技術を全て駆使して作成する事を目標としました。
最終的にはプランナーの諸事情で中止する事になり、完成には至りませんでしたが、インゲームのみのトライアル版はバンタンの発表会で高い評価を受ける事ができました。
# アピールポイント
DOTweenによるスキルのアニメーションや、キャラクターのコマ送りアニメーションを2D AnimationのSprite LibraryとSprite Solverを使用して作成し、変更に強い仕組みにしました。
