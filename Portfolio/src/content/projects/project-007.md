## overview
バンタン1年次前期の発表会のために制作した、サバイバルシミュレーションゲーム。
選択肢に応じてアイテムや道具を入手して、体力やスタミナなどのリソースを管理をしながら進み続けるゲームです。
# 説明
アドベンチャーゲームとしてのノベルシステムと、シミュレーションのリソース管理システムを研究する事を目標に制作しました。
最終的にシステム部分の完成と共に制作期間が終了したためストーリーは未完となりましたが、このシステム設計を利用して今後更なるシミュレーションゲームに役立てたいと思います。
# アピールポイント
UI ToolKitを使用した近代的なUIシステムを使用し、UIをモバイル向けに対応する事にも挑戦しました。
